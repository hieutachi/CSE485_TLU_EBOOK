# Ebook Tự Học: Lập trình Backend với PHP & Laravel

> **Môn học:** CSE485 – Công nghệ Web  
> **Đối tượng:** Sinh viên ngành Công nghệ thông tin  
> **Mức độ:** Từ số 0 đến Admin Dashboard CRUD 2 bảng với Laravel

---

## Lời mở đầu

Chào các bạn sinh viên,

Nếu bạn từng mở một trang web, điền form đăng ký, nhấn **Gửi**, rồi thấy thông báo *"Đăng ký thành công"* — bạn đã chạm vào **Backend** mà không biết. Phần "mặt tiền" đẹp đẽ trên trình duyệt là Frontend. Phần **não bộ** phía sau — nhận dữ liệu, kiểm tra, lưu vào cơ sở dữ liệu, trả kết quả — chính là Backend.

Cuốn Ebook này được soạn theo tinh thần **tự học có lộ trình**: không nhảy cóc vào Laravel ngay từ buổi đầu, mà đi từ PHP thuần → OOP → PDO → MVC mini → rồi mới Laravel. Lý do rất đơn giản:

> Framework là công cụ. Nếu bạn hiểu bản chất, Laravel sẽ là bạn đồng hành.  
> Nếu bạn chỉ "copy theo tutorial", Laravel sẽ là hộp đen.

### Mục tiêu xuyên suốt 12 buổi

| Giai đoạn | Buổi | Bạn sẽ đạt được |
|-----------|------|-----------------|
| PHP thuần | 1–4 | Viết script PHP, OOP cơ bản, CRUD MySQL bằng PDO |
| Kiến trúc | 5 | Tự build MVC mini từ mã thuần |
| Laravel | 6–10 | Route, Blade, Migration, Eloquent, Relationship |
| Dự án | 11–12 | Admin Dashboard CRUD đầy đủ giữa 2 bảng |
| Project nhóm | 13–15 | Backend hoàn chỉnh theo tiêu chí điểm B / điểm Cộng |

### Cách dùng Ebook này hiệu quả nhất

1. **Đọc lý thuyết** → vẽ lại sơ đồ ASCII bằng tay (không chỉ nhìn).
2. **Gõ lại toàn bộ code** — đừng copy-paste. Cơ bắp nhớ lệnh quan trọng hơn mắt nhìn.
3. **Làm Phiếu học tập** tương ứng (Phần 2) ngay sau mỗi buổi hoặc cụm buổi.
4. **Nộp bài đúng quy định:** GitHub Repo riêng + video OBS có camera mặt.
