# CSE485 — Ebook tự học Backend (PHP & Laravel)

Tài liệu hướng dẫn tự học môn **Công nghệ Web / Lập trình Backend** cho sinh viên TLU.

## Nội dung

Toàn bộ tài liệu nằm trong thư mục [`ebook/`](./ebook/README.md):

- **12 buổi** lý thuyết–thực hành (PHP thuần → Laravel Admin Dashboard)
- **Hướng dẫn Project cuối môn** (buổi 13–15, làm việc nhóm)

Phiếu học tập (worksheets) được giảng viên phát hành riêng, bản gọn hơn — không nằm trong repo này.

## Lưu ý quan trọng cho sinh viên

- Repo này là **hướng dẫn chi tiết** theo lộ trình, **không phải** project code mẫu hoàn chỉnh để clone rồi nộp.
- Các buổi học giải thích khái niệm, luồng xử lý, cấu trúc thư mục và cách làm từng bước.
- Khi làm phiếu / bài tập: **tự viết mã nguồn**, đẩy lên **GitHub Repository riêng**, nộp kèm video minh chứng theo yêu cầu giảng viên.
- Không copy-paste để nộp bài; hãy đọc hướng dẫn, tự gõ code, chạy được rồi mới nộp.

## Bắt đầu học

1. Mở [`ebook/README.md`](./ebook/README.md) để xem mục lục.
2. Đọc lần lượt từ [Lời mở đầu](./ebook/00-loi-mo-dau.md) → Buổi 1 → …
3. Làm phiếu học tập theo lịch / tài liệu riêng của giảng viên.
